# OMAD Quest Malaysia

An installable web app for OMAD and intermittent fasting with XP, streaks, and a customizable Malaysia-style food budget.

## Share It With Other People

Upload these files and folders to any static website host:

- `index.html`
- `manifest.webmanifest`
- `service-worker.js`
- `icons/`

Easy hosting options:

- Netlify: drag this project folder into Netlify Drop.
- Vercel: import the folder as a static project.
- GitHub Pages: push these files to a repository and enable Pages.

After it is online, people can open the link in their browser. On phones, they can use **Add to Home Screen** to install it like an app.

## Notes

The app saves each person's progress in their own browser using local storage. It does not need a login or database.
